Simplescript Engine

It's as simple as adding

<script src="dt-is-not-available.github.io/sse/stable.js"></script>

to your page

Javascript

Add new commands

SSE.cmds.test = function(arg1, arg2){}

Create new script instance

let script = SSE()

Set script variables

script.vars.xyz = 123

Run a script (lineIndex is optional)

script.run("script string", ?lineIndex)

Simplescript

Run a command

test "arg1", "arg2"

Reference a variable

test %varname%

Acceptable types

The script will accept any valid JSON type as an argument
