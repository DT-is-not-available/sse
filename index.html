Simplescript Engine

It's as simple as this:

<script src="dt-is-not-available.github.io/sse/stable.js"></script>
What is this?

This is a simple and lightweight library that creates a safe and extremely simple scripting language. You create the commands and keywords and Simplescript does the rest.

Basic use

Javascript

Add new commands

SSE.cmds.test = function(arg1, arg2){}

Add new keywords

SSE.keywords.test = function(arg1, arg2){}

Add per-line evaluation

SSE["*"] = function(line){}

Create new script instance

let script = SSE()

Set script variables

script.vars.xyz = 123

Run a script (lineIndex is optional)

script.run("script string", ?lineIndex)

Simplescript

Run a command

test "arg1", "arg2"

Run a keyword

test arg1 arg2

Reference a variable

test %varname%

Acceptable types

The script will accept any valid JSON type as an argument
